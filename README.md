# UniConverter
Calculation and conversion app.    
Min API - 26    
Locales - English / Ukranian      
           
Aveliable for converssion: 
- Area   
- Force 
- Time 
- Length 
- Temperature 
- Speed   
- Volume    
- Weight  
     
Aveliable calculators:  
- Math  
- Currency (require Internet connection)
